## Introducción
En este repositorio se encuentran las funciones primitivas recursivas necesarias para la función de potencia.

## Uso
Primero debemos compilar el programa
```bash
$ cd build
$ cmake ..
$ make
```
Una vez compilado, podemos ejecutar el programa
```bash
$ cd ../bin
$ ./main
```

En el programa se nos preguntará por a y b y reallizará la operación a^b.
